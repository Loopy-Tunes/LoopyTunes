Detailed Description

Class name: EncoderDriver Function: Driver for the encoder input, controls the value of FX parameters

Based off of Encoder class from libDaisy (Stephen Hensley, 2019) https://github.com/electro-smith/libDaisy/blob/master/src/hid/encoder.h
